# Widerruf & Kontakt

Kostenloses WordPress-Plugin mit datenschutzkonformem **Widerrufsformular** und **Kontaktformular** als native Gutenberg-Blöcke.

## Download & Anleitung

**→ [gutenblock.com/widerruf-kontakt](https://gutenblock.com/widerruf-kontakt)** — Landingpage mit Download, Features, rechtlichem Hintergrund und Schritt-für-Schritt-Anleitung.

Direkt-Download (neueste Version):

[widerruf-kontakt.zip](https://github.com/hjherbst/widerruf-kontakt/releases/latest/download/widerruf-kontakt.zip)

## Features

- Widerrufsformular für Waren, digitale Produkte und Dienstleistungen
- Kontaktformular mit konfigurierbaren Feldern
- Eingangsbestätigung per E-Mail an den Absender
- SMTP-Wizard (Brevo, Postfach, manuell) inkl. Test-Mail
- Admin-Anleitung mit Schnellstart „Widerrufsseite erstellen"
- Honeypot + IP-Rate-Limit-Spamschutz
- Keine Datenbankprotokollierung (DSGVO-Datensparsamkeit)
- Auto-Updates über GitHub Releases
- EN-Default, DE auf deutschen Sites

## Anforderungen

- WordPress 6.3+
- PHP 7.4+

## Installation

1. ZIP von der [Landingpage](https://gutenblock.com/widerruf-kontakt) oder [Releases](https://github.com/hjherbst/widerruf-kontakt/releases) herunterladen
2. WordPress Admin → Plugins → Installieren → Plugin hochladen → Aktivieren
3. Admin-Menü **Widerruf & Kontakt** → Anleitung folgen
